Senior Design Project

My Senior Design project is a web platform for Java specifications. What this means is that I am working to create a web platform to create specifications for common Java libraries. This is being done with a social and technical side. The social side is using a Question and Answer site to automatically post questions regarding specifications. We then retrieve and analyse the answers to create a better specification.

We are using Java and HTTP requests to communicate with a Question and Answer site to get specifications as well as to analyze them.

My role on the team is web master and key concept holder. This means that I manage the website for the team (listed below) and have in depth knowledge on how our project is composed.

  • Graph Theory (ComS 311 Project 2)

    • This project was focused primarily on the implementation and traversal of graphs. We were to set up a weighted graph representing the city of Ames.       Overall I learned a huge amount about graph theory by completing this project. I had taken graph theory courses before but this was the first time that I had really been able to implement the theory and see it work on a larger scale (upwards of 1500 vertices).
